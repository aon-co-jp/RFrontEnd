# RFrontEnd

Un dépôt parent qui regroupe plusieurs projets développant des équivalents de
HTML5/CSS3/TypeScript/React entièrement à partir de zéro, sans jamais
réutiliser le code d'implémentations existantes.

📖 Autres langues : [日本語](README-Japan.md) / [English](README-English.md) /
[中文](README-Chinese.md) / [한국어](README-Korea.md) / [Español](README-Spain.md) /
[Français](README-France.md) / [Deutsch](README-Germany.md) / [Italiano](README-Italy.md) /
[Русский](README-Russia.md) / [العربية](README-Arabic.md) —
Pour intégrer ce projet ailleurs, consultez **[PORTING.md](PORTING.md)**,
un fichier autonome unique.

## Projets subordonnés

| Projet | Rôle | Dépôt |
|---|---|---|
| RHTML | Analyseur/implémentation DOM équivalent à HTML5 | [RTHML](https://github.com/aon-co-jp/RTHML) |
| RCSS | Analyseur/calcul de cascade équivalent à CSS3 | [RCSS](https://github.com/aon-co-jp/RCSS) |
| RTypeScript | Équivalent de TypeScript | [RTypeScript](https://github.com/aon-co-jp/RTypeScript) |
| RJSON | Traitement JSON | [RJSON](https://github.com/aon-co-jp/RJSON) |
| RReact | Modèle de composants équivalent à React | [RReact](https://github.com/aon-co-jp/RReact) |

L'environnement d'exécution côté serveur est pris en charge par
[RPoem](https://github.com/aon-co-jp/RPoem) (Poem redéveloppé de zéro) et
[RCosmo](https://github.com/aon-co-jp/RCosmo) (anciennement open-runo).

Consultez `CLAUDE.md` pour la politique de développement détaillée, l'ordre
de construction et l'architecture.
