# RFrontEnd

Un repository padre che raggruppa più progetti che sviluppano equivalenti di
HTML5/CSS3/TypeScript/React interamente da zero, senza riutilizzare in alcun
modo il codice di implementazioni esistenti.

📖 Altre lingue: [日本語](README-Japan.md) / [English](README-English.md) /
[中文](README-Chinese.md) / [한국어](README-Korea.md) / [Español](README-Spain.md) /
[Français](README-France.md) / [Deutsch](README-Germany.md) / [Italiano](README-Italy.md) /
[Русский](README-Russia.md) / [العربية](README-Arabic.md) —
Per integrare questo progetto altrove, consulta **[PORTING.md](PORTING.md)**,
un unico file autonomo.

## Progetti subordinati

| Progetto | Ruolo | Repository |
|---|---|---|
| RHTML | Parser/implementazione DOM equivalente a HTML5 | [RTHML](https://github.com/aon-co-jp/RTHML) |
| RCSS | Parser/calcolo della cascata equivalente a CSS3 | [RCSS](https://github.com/aon-co-jp/RCSS) |
| RTypeScript | Equivalente di TypeScript | [RTypeScript](https://github.com/aon-co-jp/RTypeScript) |
| RJSON | Elaborazione JSON | [RJSON](https://github.com/aon-co-jp/RJSON) |
| RReact | Modello a componenti equivalente a React | [RReact](https://github.com/aon-co-jp/RReact) |

L'ambiente di esecuzione lato server è gestito da
[RPoem](https://github.com/aon-co-jp/RPoem) (Poem risviluppato da zero) e
[RCosmo](https://github.com/aon-co-jp/RCosmo) (ex open-runo).

Per la politica di sviluppo dettagliata, l'ordine di costruzione e
l'architettura, consulta `CLAUDE.md`.
