# RFrontEnd

مستودع أب يجمع عدة مشاريع تُطوِّر ما يعادل HTML5/CSS3/TypeScript/React من
الصفر تمامًا، دون إعادة استخدام أي كود من التطبيقات الحالية.

📖 لغات أخرى: [日本語](README-Japan.md) / [English](README-English.md) /
[中文](README-Chinese.md) / [한국어](README-Korea.md) / [Español](README-Spain.md) /
[Français](README-France.md) / [Deutsch](README-Germany.md) / [Italiano](README-Italy.md) /
[Русский](README-Russia.md) / [العربية](README-Arabic.md) —
لدمج هذا المشروع في مشروع آخر، راجع **[PORTING.md](PORTING.md)**، وهو ملف
واحد مكتفٍ بذاته.

## المشاريع التابعة

| المشروع | الدور | المستودع |
|---|---|---|
| RHTML | محلل/تطبيق DOM مكافئ لـ HTML5 | [RTHML](https://github.com/aon-co-jp/RTHML) |
| RCSS | محلل/حساب تتابع (cascade) مكافئ لـ CSS3 | [RCSS](https://github.com/aon-co-jp/RCSS) |
| RTypeScript | مكافئ لـ TypeScript | [RTypeScript](https://github.com/aon-co-jp/RTypeScript) |
| RJSON | معالجة JSON | [RJSON](https://github.com/aon-co-jp/RJSON) |
| RReact | نموذج مكونات مكافئ لـ React | [RReact](https://github.com/aon-co-jp/RReact) |

بيئة التشغيل من جانب الخادم يتولاها
[RPoem](https://github.com/aon-co-jp/RPoem) (إعادة تطوير Poem من الصفر) و
[RCosmo](https://github.com/aon-co-jp/RCosmo) (المعروف سابقًا باسم
open-runo).

راجع `CLAUDE.md` للاطلاع على سياسة التطوير التفصيلية وترتيب البناء
والبنية المعمارية.
