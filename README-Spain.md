# RFrontEnd

Un repositorio padre que agrupa varios proyectos que desarrollan equivalentes
de HTML5/CSS3/TypeScript/React completamente desde cero, sin reutilizar en
absoluto el código de implementaciones existentes.

📖 Otros idiomas: [日本語](README-Japan.md) / [English](README-English.md) /
[中文](README-Chinese.md) / [한국어](README-Korea.md) / [Español](README-Spain.md) /
[Français](README-France.md) / [Deutsch](README-Germany.md) / [Italiano](README-Italy.md) /
[Русский](README-Russia.md) / [العربية](README-Arabic.md) —
Para integrar este proyecto en otro, consulta **[PORTING.md](PORTING.md)**,
un único archivo autocontenido.

## Proyectos subordinados

| Proyecto | Rol | Repositorio |
|---|---|---|
| RHTML | Analizador/implementación de DOM equivalente a HTML5 | [RTHML](https://github.com/aon-co-jp/RTHML) |
| RCSS | Analizador/cálculo de cascada equivalente a CSS3 | [RCSS](https://github.com/aon-co-jp/RCSS) |
| RTypeScript | Equivalente a TypeScript | [RTypeScript](https://github.com/aon-co-jp/RTypeScript) |
| RJSON | Procesamiento de JSON | [RJSON](https://github.com/aon-co-jp/RJSON) |
| RReact | Modelo de componentes equivalente a React | [RReact](https://github.com/aon-co-jp/RReact) |

El entorno de ejecución del lado del servidor está a cargo de
[RPoem](https://github.com/aon-co-jp/RPoem) (Poem redesarrollado desde cero)
y [RCosmo](https://github.com/aon-co-jp/RCosmo) (antes open-runo).

Consulta `CLAUDE.md` para conocer en detalle la política de desarrollo,
el orden de construcción y la arquitectura.
